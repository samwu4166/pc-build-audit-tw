# PC Build Audit — h12-curveball-china-grey-market

**Persona**: 38 歲老粉，菜單有水貨 X3D + 中國貨，問值不值
**預算**: NT$ 38,000
**Use case 推測**: AM4 末代信仰升級，gaming 1080p/1440p（X3D 招牌就是吃 cache 的遊戲）。若是 AI/LLM dev 這菜單**完全不對**（8GB VRAM 直接判出局），audit 以 gaming 為主軸。

---

## 🚨 必修 BLOCKER（高度警告，但不是不能用）

### 1. ⚠️ 水貨 5800X3D + 水貨 4060 Ti — 雙水貨疊加風險爆表

**核心問題：保固 territory limit + reseller 跑路風險**

- **AMD 水貨 CPU**：AMD 全球三年保但**台灣 AMD 不收**水貨 RMA。CPU 算耐用品還算可接受，但 5800X3D 是末代 AM4 旗艦，**故障了你要寄回中國/美國通路**，運費+時間成本破表。
- **索泰 4060 Ti 水貨更慘**：索泰台灣代理（捷元/捷修網）**完全不收水貨**。卡掛了 = 整張變紙鎮。GPU 比 CPU 更容易掛（風扇軸承、VRAM、電容）。**4060 Ti 8GB 二手脫手率本身就低**（VRAM 不夠是 2026 共識），加上水貨無保 = 賣不掉。
- **2026-06 台灣 5800X3D 行情**：公司貨已停產缺貨，二手 $6,500-7,500（PTT），水貨報 $5,500 **省的不到 $2k**，但保固消失 = 不值。

**Fix path A（推薦）**：CPU 改公司貨二手 5800X3D $6,500-7,000（PTT PC_Shopping 買賣版有），GPU 改**公司貨 RTX 5060 Ti 16GB $13,500-14,500**（雙倍 VRAM、新世代、5 年代理保、AV1 編碼）。整機預算還壓得住 38K。

**Fix path B（不推薦但合理）**：認賠保固，當「3 年內不壞就賺到」的賭注。但 4060 Ti 8GB 在 2026 已經是上一代尷尬卡（1440p 高畫質吃緊），**現金價也撐不住**。

### 2. ⚠️ B450M Steel Legend × 5800X3D — BIOS 必須先更新

B450 板原生不支援 Zen 3，需 BIOS **P5.50 以上**才能點亮 5800X3D。**ASRock B450M Steel Legend 沒有 BIOS FlashBack 按鈕**，沒有可開機 CPU 不能刷 BIOS。

**Fix path A**：買 MB 時跟賣家明講「要先刷到最新 BIOS 才出貨」（原價屋/欣亞通常免費代刷）。
**Fix path B**：改 **B550M Steel Legend / B550M Pro4 $2,800-3,200**，原生支援 5800X3D 免刷 BIOS，PCIe 4.0 全開（B450 只有 GPU 槽 Gen3，省這 $500 不划算）。

### 3. ⚠️ 機殼 + 風扇沒列 — 預算缺口至少 $1,500-2,500

菜單**沒有機殼**也沒列額外風扇。FX240 AIO 通常只附 2 顆 240 排風扇，機殼進氣風扇要另買。預算要再扣 $1,500-2,500。

## 📋 PSU 瓦數驗算（必印）

```
5800X3D TDP = 105W
RTX 4060 Ti TDP = 160W
required = (105 + 160) × 1.3 = 344.5W → 推 550W~650W
```

**DQ650-M 650W gold 瓦數 OK，但**：
- DeepCool DQ 系列是 2018 老設計，**非 ATX 3.0**（沒 12V-2x6，但 4060 Ti 用 8pin 不影響）。
- 5800X3D + 4060 Ti 平台耐用沒問題，**過剩但安全**。
- 若未來想升 4070 Super / 5070（200W+）這顆吃緊，建議直接上 **MSI MAG A650GL 650W ATX 3.0 $2,290** 或 **海韻 Focus GX-650 $2,990**（10 年保）。

## 🟡 2026 SSD / RAM HBM 紅利期 baseline（強制 callout）

> 因 HBM3e + AI 訓練卡需求佔走 NAND / DRAM 產能，2026 SSD / DDR5 比 2023 漲 30-50%。
> - **DDR4** 反而因 AM4 復甦+產能轉移 **漲更兇**（PTT #1cZxxxxx 12 月討論 DDR5 漲到逼 DDR4 用戶留守 AM4）。DDR4-3600 32GB 阿斯加特 ~$2,200-2,800 2026 是合理區間。
> - **Patriot P400 2TB Gen4** ~$3,200-3,800 合理；低於 $2,800 反而要懷疑是 P400 Lite（DRAM-less，QLC，掉速嚴重，跟 P400 不同產品）。**這菜單必須確認是 P400 還是 P400 Lite**。

## 📊 Use-case 適配度

- **Primary use（推測）**: 1080p/1440p gaming（X3D 招牌）
- **適配亮點（改版後）**:
  - **5800X3D + 5060 Ti 16GB**：1440p 高畫質主流 3A 60-100fps（cyberpunk DLSS、艾爾登法環、TLOU2）
  - **16GB VRAM** 撐 2026-2028 遊戲世代（8GB 已被 Indiana Jones / Hogwarts Legacy 用爆）
  - **32GB DDR4-3600** + 2TB Gen4 SSD 對 gaming + 平常用足
- **不適合**：
  - **AI / LLM dev**：5800X3D 沒 AVX-512、單 GPU 16GB 跑 Llama 3.1 8B Q4 可以但 13B 吃緊，認真做 dev 直上 AM5 + 5070 Ti
  - **直播編碼**：5060 Ti NVENC 第 9 代很強，OK 但 CPU 8 核已是上限
- **AM4 升級空間**: 🪦 **死路一條**。5800X3D 是 AM4 終極 CPU，未來只能換整套 AM5。RAM/SSD 之後搬不過去（DDR4→DDR5 不相容）。**這是純「用 5 年榨乾」的決策**。

## ⚠️ 還沒解的 risk

1. **5800X3D 公司貨二手取得**：PTT PC_Shopping 買賣版近 1 個月有 3-5 筆 $6,500-7,500，**要看面交驗機**；不熟二手交易就回到水貨/或預算拉到 40K 等公司貨補貨。
2. **B450M Steel Legend 不是不能用**：硬要省 $500 留 B450M，**務必要求賣家代刷 BIOS 到 P5.50+**，否則買回家黑屏無解（沒 FlashBack）。
3. **DDR4-3600 在 B450/B550 不一定吃**：AM4 IMC 甜蜜點 3200-3466，3600 看體質。實際可能跑 3466 + FCLK 1733 反而最穩。
4. **Patriot P400 vs P400 Lite**：菜單寫 P400，要確認賣家不是給 Lite 版（價差 $1,000，效能差 50%）。
5. **8GB 4060 Ti 賭注**：若 2-3 年後 1440p 8GB VRAM 全面爆（已開始），這張卡 wave 一波就脫手不掉。改 5060 Ti 16GB 多 $5k 是保險，不是浪費。
6. **DDR4-3600 阿斯加特**：中國品牌台灣保固薄，跑了 EXPO/XMP 不穩 RMA 麻煩。**改十銓/威剛/金士頓多花 $200-300 換終身保**。

## 🎯 老粉決策建議（直話直說）

**38 歲老粉信仰 5800X3D 我懂，但要分兩派**：

**派 A — 純信仰末代王**：上面 fix path A（公司貨二手 5800X3D + 5060 Ti 16GB + B550M）。**這套是 AM4 退場最後體面**，玩 5 年榨乾，2030 再砍掉重練 AM6 / Zen 7。

**派 B — 預算硬卡 38K 不能多**：放棄 5800X3D，直上 **R5 7500F + B650M + DDR5-6000 32GB + 5060 Ti 16GB**（AM5 入門盤）。約 $36-37K 可達，未來 3-5 年還有 CPU 升級空間（Zen 5 X3D / Zen 6）。**這是理性派但放棄信仰**。

水貨那套 $38K 看起來划算，**3 年後 GPU 掛掉維修無門 + 8GB VRAM 賣不掉**，實際 TCO 比改版貴。**水貨不是不能買，是不能 CPU+GPU 雙水貨疊加**。
