# PC Build Audit — case4-fatal-compat-bug

> 預算 NT$ 35,000｜使用者：40 歲老父，聽兒子建議從原價屋複製菜單

## 🚨 必修 BLOCKER（連開都開不了機，先別下訂）

你這份菜單有 **兩個致命相容性錯誤**，照原單買回家組起來會直接點不亮，錢就丟水裡。

### BLOCKER #1：CPU 跟主機板**腳位不同平台**

- **AMD R7 7700** = AM5 腳位（AMD 平台）
- **ASUS PRIME B760M-A D4** = LGA 1700 腳位（**Intel** 平台，B760 是 Intel chipset）
- 物理上 CPU 根本**插不進去**主機板，Intel 板配 AMD CPU = 完全不能用
- 這是 PTT PC_Shopping 半年至少出現 3 次的經典悲劇

### BLOCKER #2：記憶體世代錯（即便 #1 修了也不能用）

- **AM5 平台只吃 DDR5**，不吃 DDR4
- 你選的「威剛 DDR4-3600」是 DDR4，插上 AM5 板**插槽卡榫位置不同**，物理上也插不進去
- 兒子可能是看到 `B760M-A D4` 的「D4」誤以為要配 DDR4，那是 Intel 板配 DDR4 沒錯，但 AMD AM5 沒這選項

### 兩條修法（建議走 Path A，預算內可達成）

**Path A：保留 R7 7700（AMD 平台），換掉主機板 + RAM** ← 推薦

| 換掉 | 換成 | 增減 |
|---|---|---|
| ASUS PRIME B760M-A D4 | ASUS PRIME B650M-A WiFi（AM5 + DDR5）| ~+$500 |
| 威剛 DDR4-3600 32GB | 威剛 XPG DDR5-6000 32GB CL30 EXPO | ~+$500 |

**Path B：保留 B760M-A D4 主機板（Intel 平台），換 CPU**

| 換掉 | 換成 |
|---|---|
| AMD R7 7700 | Intel Core i5-14400F（LGA1700 + DDR4 都吃）|

→ Path B 的 i5-14400F 多核效能稍弱於 R7 7700，但便宜約 $2000，預算更鬆。

## 📊 Use-case 適配度

兒子幫老爸選的這組，看起來是 **「一般家用 + 偶爾 1080p 遊戲」** 路線，沒有特殊需求（不跑 AI、不剪片、不直播）。

- **文書 / 上網 / YouTube**：R7 7700 + 32GB RAM 過剩，5 年不會卡
- **1080p 遊戲**：RX 7600 8GB 跑《艾爾登法環》《俠盜獵車手 V》中高畫質 60fps 順
- **1440p 遊戲**：勉強，部分 3A 大作要降特效
- **AI / LLM**：❌ 8GB VRAM 跑不動 Llama 8B（Q4 需 ~6GB + overhead），如果老爸/兒子有想玩本地 AI，要升 RX 7800 XT 16GB 或 RTX 5060 Ti 16GB（預算會破 $40k）
- **未來升級**：AM5 平台戰到 2027+，CPU 可換 9700X / 9800X3D；RAM 2 槽空著可加到 64GB

## ⚠️ 還沒解的 Risk

1. **PSU CV650 非模組化**：原單留著的話，背線會凌亂，老爸自組可能塞不漂亮；上門組裝師傅幫塞 OK
2. **Antec NX410 預裝風扇是非 ARGB 黑扇**：不在意外觀沒差，想要 RGB 要另外買
3. **B650M-A WiFi 是 mATX**：板小、PCIe 槽少，未來要加擷取卡 / 第二顆 GPU 沒空間（但這 build 用不到）
4. **DDR5-6000 EXPO 開機要進 BIOS 開**：預設跑 4800，老爸自己開機可能不知道要進 BIOS 開 EXPO；建議叫上門組裝師傅開好
5. **保固分散**：原價屋零組件各廠保固獨立，整機壞要自己拆出哪個零件壞掉 → 老爸如果不熟電腦，**強烈建議走原價屋上門組裝 + 一年整機保固**
